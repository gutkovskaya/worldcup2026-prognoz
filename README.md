# ЧМ-2026 — прогнозы и аналитика

Живой HTML-отчёт с прогнозами плей-офф Чемпионата мира по футболу 2026.

- **Сайт:** https://worldcup2026-prognoz.netlify.app
- **Деплой:** Netlify, git-based — каждый `git push` в этот репозиторий автоматически пересобирает сайт.
- Единственный файл — `index.html` (самодостаточный, без внешних зависимостей).

Прогнозы строятся по 6 факторам: форма, тренер, судья, место проведения, стабильность состава, травмы.
